# Project Setup

This document lists all the libraries and dependencies installed for this React project.

## Installed Libraries

### Dependencies

- **axios** (^1.13.2) - Promise-based HTTP client for making API requests
- **plotly.js** (^3.3.0) - JavaScript graphing library for creating interactive charts and graphs
- **react** (^19.2.0) - JavaScript library for building user interfaces
- **react-dom** (^19.2.0) - React renderer for DOM operations
- **react-plotly.js** (^2.6.0) - React wrapper for plotly.js
- **react-router-dom** (^7.9.6) - Declarative routing for React applications

### Dev Dependencies

- **@eslint/js** (^9.39.1) - ESLint JavaScript plugin
- **@types/react** (^19.2.2) - TypeScript type definitions for React
- **@types/react-dom** (^19.2.2) - TypeScript type definitions for react-dom
- **@vitejs/plugin-react** (^5.0.0) - Official Vite plugin for React
- **autoprefixer** (^10.4.22) - PostCSS plugin to parse CSS and add vendor prefixes
- **eslint** (^9.39.1) - JavaScript linter for identifying and reporting code issues
- **eslint-plugin-react-hooks** (^7.0.1) - ESLint rules for React Hooks
- **eslint-plugin-react-refresh** (^0.4.24) - ESLint plugin for React Refresh
- **globals** (^16.5.0) - Global variables for ESLint
- **postcss** (^8.5.6) - Tool for transforming CSS with JavaScript plugins
- **tailwindcss** (^4.1.17) - Utility-first CSS framework
- **vite** (^7.2.2) - Next generation frontend build tool

## Project Structure

```
src/
├── components/    # Reusable React components
├── layout/        # Layout components
├── pages/         # Page components
├── hooks/         # Custom React hooks
├── styles/        # Additional stylesheets
├── App.jsx        # Main App component
├── index.css      # Global styles (includes Tailwind directives)
└── main.jsx       # Application entry point
```

## Configuration Files

- **tailwind.config.js** - Tailwind CSS configuration
- **postcss.config.js** - PostCSS configuration for Tailwind CSS
- **vite.config.js** - Vite build tool configuration
